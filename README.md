# Проект: Управление категориями и продуктами
## Описание функционала

- В рамках текущего задания реализованы две основные сущности — Product и Category.

- Класс Product описывает товар с такими атрибутами, как название, описание, цена и количество на складе.

- Класс Category описывает категорию товаров с названием, описанием и списком продуктов.

- В классе Category реализованы статические атрибуты для подсчёта общего количества категорий и товаров, которые автоматически обновляются при создании новых объектов.

- Добавлена функция загрузки данных из JSON-файла, позволяющая создавать объекты категорий и продуктов из внешнего файла.

- Реализованы тесты, проверяющие корректность инициализации объектов и подсчёт статистики.
