# Проект: Управление категориями и продуктами
## Описание функционала

- В рамках текущего задания реализованы две основные сущности — Product и Category.

- Класс Product описывает товар с такими атрибутами, как название, описание, цена и количество на складе.

- Класс Category описывает категорию товаров с названием, описанием и списком продуктов.

- В классе Category реализованы статические атрибуты для подсчёта общего количества категорий и товаров, которые автоматически обновляются при создании новых объектов.

- Добавлена функция загрузки данных из JSON-файла, позволяющая создавать объекты категорий и продуктов из внешнего файла.

- Для расширения функционала реализованы два класса-наследника:
  - `Smartphone` — добавлены атрибуты производительность (`efficiency`), модель (`model`), объем памяти (`memory`) и цвет (`color`).
  - `LawnGrass` — добавлены атрибуты страна-производитель (`country`), срок прорастания (`germination_period`) и цвет (`color`).

- Доработана операция сложения для продуктов: складывать можно только объекты одного класса. При попытке сложения объектов разных классов выбрасывается ошибка `TypeError`.

- Метод `add_product` в категории теперь проверяет тип добавляемого объекта: можно добавлять только продукты или их наследников. Попытка добавить другой объект вызывает `TypeError`.

- Реализованы тесты, проверяющие корректность новой функциональности и подсчёт статистики. Покрытие тестами функционального кода более 75%.
